JavaScript Fundamentals

JavaScript es un lenguaje multiparadigma, dinámico y basado en prototipos, ampliamente utilizado para desarrollo web. A diferencia de Python, JS corre principalmente en el navegador y está profundamente ligado al manejo del DOM.

Historia

JavaScript fue creado en 1995 por Brendan Eich en Netscape en solo 10 días. Inicialmente diseñado para scripts en el navegador, hoy es uno de los lenguajes más usados gracias a entornos como Node.js.

Variables

JavaScript tiene tres formas principales de declarar variables:

  • var: alcance global o funcional, permite redeclaración.
  • let: alcance de bloque, mutable.
  • const: alcance de bloque, no se puede reasignar. 
var a = 10;
let b = 20;
const c = 30;

Funciones

function suma(a, b) {
  return a + b;
}

const resta = (a, b) => a - b;

Scope

El scope define la accesibilidad de las variables. En JS existen: global, de función y de bloque.

Hoisting

JavaScript eleva declaraciones al inicio de su contexto.

console.log(x); // undefined
var x = 5;

Closures

Un closure es una función que recuerda el entorno en el que fue creada. 

function contador() {
  let count = 0;
  return function() {
    count++;
    return count;
  }
}

Callbacks

function procesar(callback) {
  callback();
}

Promesas

const promesa = new Promise((resolve, reject) => {
  setTimeout(() => resolve("OK"), 1000);
});

Async / Await

async function obtenerDatos() {
  const res = await fetch('https://api.example.com');
}
