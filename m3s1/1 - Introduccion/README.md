# 📘 JavaScript Fundamentals — Introducción

## 🎯 Objetivo

Este módulo introduce los fundamentos esenciales de JavaScript como lenguaje de programación y como herramienta clave en el desarrollo frontend.

Está diseñado para personas que ya tienen bases en programación (por ejemplo, Python) y conocimientos previos de HTML y CSS, buscando entender cómo JavaScript aporta dinamismo, asincronía y control sobre el navegador.

---

## 🧠 Enfoque

A diferencia de un curso introductorio superficial, este módulo busca:

* Comprender **cómo funciona JavaScript internamente**
* Identificar diferencias clave con lenguajes como Python
* Construir una base sólida para escalar hacia aplicaciones más complejas (SPA)

## 📖 Contenido

El archivo `index.html` incluye una introducción teórica acompañada de ejemplos prácticos sobre:

### 🔹 Fundamentos del lenguaje

* Historia de JavaScript
* Naturaleza del lenguaje (dinámico, interpretado, multiparadigma)

### 🔹 Variables

* `var`, `let`, `const`
* Diferencias de scope y mutabilidad

### 🔹 Funciones

* Declarativas
* Arrow functions

### 🔹 Scope

* Global
* Funcional
* De bloque

### 🔹 Hoisting

* Elevación de variables
* Diferencias entre `var`, `let` y `const`

### 🔹 Closures

* Persistencia del contexto léxico
* Uso práctico

### 🔹 Callbacks

* Funciones como argumentos
* Base de la asincronía

### 🔹 Promesas

* Estados (`pending`, `resolved`, `rejected`)
* Manejo con `.then()`

### 🔹 Async / Await

* Sintaxis moderna para asincronía
* Manejo más legible de promesas

---

## ⚙️ Ejecución

1. Abre el archivo `index.html` en el navegador
2. Abre la consola del navegador (F12)
3. Haz clic en el botón **"Ejecutar ejemplos"**
4. Observa los resultados en consola
