# 🚀 JavaScript Fundamentals

## 🎯 Objetivo

Este repositorio tiene como objetivo enseñar JavaScript desde cero hasta la construcción de una **Single Page Application (SPA)**, utilizando únicamente JavaScript vanilla.

El enfoque está en comprender:

* Cómo funciona el lenguaje internamente
* Cómo interactúa con el navegador
* Cómo se construyen aplicaciones frontend reales

---

## 🧠 Filosofía

Este proyecto **no es un curso de copiar y pegar código**.

Busca desarrollar:

* Pensamiento lógico
* Comprensión profunda del lenguaje
* Buenas prácticas de frontend

> ⚠️ La meta no es que “funcione”, sino que entiendas **por qué funciona**.

## 📚 Contenido

### 🔹 1. Introducción

* Historia de JavaScript
* Variables (`var`, `let`, `const`)
* Funciones
* Scope
* Hoisting
* Closures
* Callbacks
* Promesas
* Async/Await

---

### 🔹 2. DOM

* Manipulación del DOM
* Eventos
* Render dinámico
* Consumo de APIs
* Flujo frontend básico

---

### 🔹 3. SPA Part 1

* Routing con hash
* Vistas dinámicas
* Render sin recarga
* Consumo de API en vistas

---

### 🔹 4. SPA Part 2

* Componentes (sin framework)
* Estado en memoria
* Persistencia (`localStorage`, `sessionStorage`)
* Composición de UI

## ⚙️ Cómo usar el proyecto

1. Clona el repositorio
2. Abre el `index.html` principal
3. Navega entre los módulos
4. Sigue los README de cada sección
5. Completa los ejercicios

---

## 🧪 Metodología

Cada módulo incluye:

* Explicación teórica
* Ejemplos prácticos
* Ejercicios progresivos (tipo bootcamp)

> ⚠️ Si no haces los ejercicios, no estás aprendiendo.

---

## 🧱 Lo que NO incluye (aún)

* Frameworks (React, Vue, Angular)
* State management avanzado
* Backend
* Testing

---

## 🧭 Roadmap sugerido

1. Completar Introducción
2. Dominar manipulación del DOM
3. Entender flujo de una SPA
4. Comprender estado y componentes
5. Migrar a framework (opcional)

---

## 🚀 Resultado esperado

Al completar este repositorio deberías ser capaz de:

* Entender cómo funciona JavaScript en el navegador
* Manipular el DOM con seguridad
* Construir una SPA básica sin frameworks
* Comprender cómo funcionan internamente React o Vue

---

## 🧠 Nota final

Aprender JavaScript sin entender estos fundamentos es construir sobre arena.

Este repositorio está diseñado para darte una base sólida.

A partir de aquí, los frameworks dejan de ser “magia”.
