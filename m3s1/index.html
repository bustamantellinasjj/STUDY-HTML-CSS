¿Qué es este proyecto?

Este repositorio es una guía progresiva para aprender JavaScript desde sus fundamentos hasta la construcción de una Single Page Application (SPA) sin frameworks.

El enfoque no es memorizar sintaxis, sino entender cómo funciona el lenguaje, cómo interactúa con el navegador y cómo escalar hacia aplicaciones reales.

Módulos

1. Introducción

Fundamentos del lenguaje: variables, funciones, scope, closures y asincronía.

2. DOM

Manipulación del DOM, eventos y consumo de APIs.

3. SPA Part 1

Routing, vistas dinámicas y estructura básica de SPA.

4. SPA Part 2

Componentes, estado y persistencia con storage.

Roadmap de aprendizaje

  • Fundamentos del lenguaje
  • Manipulación del DOM
  • Eventos y asincronía
  • Single Page Applications
  • Arquitectura frontend básica

Este recorrido te prepara para entender frameworks como React o Vue desde una base sólida.
