Estructura HTML5 Semántica

Estructura base de un archivo HTML

Todo archivo HTML empieza con esta estructura obligatoria:

<!-- Declara que es HTML5 -->
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Título de la página</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <!-- Todo el contenido visible va aquí -->
</body>
</html>

Etiquetas semánticas principales

EtiquetaPara qué sirve
<header>Encabezado de la página o sección
<nav>Menú de navegación
<main>Contenido principal de la página
<section>Sección temática del contenido
<article>Contenido independiente (tarjeta, post)
<aside>Contenido secundario o lateral
<footer>Pie de página
<div>Contenedor genérico sin significado semántico
💡 Tip: Las etiquetas semánticas le dicen al navegador QUÉ es el contenido, no solo cómo se ve. Mejoran accesibilidad y SEO.

Atributos importantes

AtributoQué haceEjemplo
idIdentificador único en la páginaid="sobre-mi"

CSS Flexbox Completo

¿Qué es Flexbox?

Flexbox organiza elementos en una fila o columna. Se activa con display: flex en el contenedor padre.

.contenedor {
  display: flex; /* activa flexbox */
}

Propiedades del CONTENEDOR (padre)

PropiedadValores comunesQué hace
displayflexActiva Flexbox
flex-directionrow | column | row-reverse | column-reverseDirección de los elementos
justify-contentflex-start | center | flex-end | space-between | space-around | space-evenlyAlineación horizontal
align-itemsflex-start | center | flex-end | stretchAlineación vertical
flex-wrapwrap | nowrapSi los elementos bajan de línea al no caber
gap10px | 1remEspacio entre elementos
.contenedor {
  display: flex;
  flex-direction: row;           /* fila (por defecto) */
  justify-content: space-between; /* espacio entre elementos */
  align-items: center;          /* centra verticalmente */
  flex-wrap: wrap;             /* bajan si no caben */
  gap: 20px;                   /* espacio entre items */
}

Propiedades de los ITEMS (hijos)

PropiedadQué hace
flex-grow: 1El item crece para ocupar el espacio disponible
flex-shrink: 0El item NO se achica aunque no haya espacio

💡 Regla fácil: justify-content = eje horizontal · align-items = eje vertical. Cuando flex-direction es column, se invierten.

CSS Grid Layout Completo

¿Qué es CSS Grid?

Grid organiza elementos en filas Y columnas al mismo tiempo. Ideal para layouts complejos como galerías y dashboards.

.contenedor {
  display: grid;
  grid-template-columns: repeat(3, 1fr); /* 3 columnas iguales */
  gap: 20px;
}

Propiedades principales del CONTENEDOR

PropiedadEjemploQué hace
display: griddisplay: gridActiva CSS Grid
grid-template-columnsrepeat(3, 1fr)Define cuántas columnas y su tamaño
grid-template-rows100px auto 200pxDefine el alto de las filas
gap20pxEspacio entre filas y columnas
column-gap20pxSolo espacio entre columnas
row-gap10pxSolo espacio entre filas
justify-itemscenterAlineación horizontal de los items
align-itemscenterAlineación vertical de los items

repeat() y fr — las unidades de Grid

/* repeat(N, tamaño): crea N columnas del mismo tamaño */
grid-template-columns: repeat(3, 1fr);  /* 3 columnas iguales */
grid-template-columns: repeat(2, 1fr);  /* 2 columnas iguales */

/* fr = fracción del espacio disponible */
grid-template-columns: 1fr 2fr 1fr;    /* col del medio el doble de ancha */
grid-template-columns: 200px 1fr;      /* sidebar fijo + contenido flexible */

Transiciones CSS

¿Qué es una transición?

Hace que un cambio de estilo ocurra suavemente en vez de de golpe. Se activa normalmente con :hover.

.elemento {
  transition: propiedad duración función-de-tiempo;

  /* Ejemplos: */
  transition: background-color 0.3s ease;
  transition: transform 0.5s ease-in-out;
  transition: all 0.3s ease; /* transiciona todo */
}

Funciones de tiempo

FunciónComportamiento
easeArranca rápido, frena al final (por defecto)
ease-inArranca lento, acelera al final
ease-outArranca rápido, frena al final
ease-in-outArranca lento, acelera en el medio, frena al final
linearVelocidad constante de inicio a fin

Ejemplos de transiciones comunes

/* 1. Cambio de color */
.boton { transition: background-color 0.3s ease; }
.boton:hover { background-color: #f0c040; }

/* 2. Escalar (agrandar) */
.tarjeta { transition: transform 0.3s ease; }
.tarjeta:hover { transform: scale(1.05); }

/* 3. Subir al hacer hover */
.tarjeta { transition: transform 0.3s ease; }
.tarjeta:hover { transform: translateY(-5px); }

/* 4. Sombra */
.caja { transition: box-shadow 0.3s ease; }
.caja:hover { box-shadow: 0 8px 20px rgba(0,0,0,0.3); }

/* 5. Borde redondeado */
.img { transition: border-radius 0.3s ease; }
.img:hover { border-radius: 50%; }

Animaciones con @keyframes

Diferencia entre transición y animación

TransiciónAnimación
Se activaCon :hover u otro eventoAutomáticamente
Se repiteNoSí (infinite)
PasosSolo inicio y finMúltiples pasos (0%, 50%, 100%)

Sintaxis de @keyframes

/* 1. Define los pasos de la animación */
@keyframes nombre-animacion {
  0%   { transform: translateY(0px); }    /* inicio */
  50%  { transform: translateY(-20px); } /* mitad */
  100% { transform: translateY(0px); }    /* fin */
}

/* 2. Aplica la animación al elemento */
.elemento {
  animation: nombre-animacion 2s ease-in-out infinite;
  /*          nombre            duración  función  repetición */
}

Animaciones comunes

/* Flotar (arriba y abajo) */
@keyframes flotar {
  0%, 100% { transform: translateY(-10px); }
  50%      { transform: translateY(10px); }
}

/* Girar */
@keyframes girar {
  0%   { transform: rotate(0deg); }
  100% { transform: rotate(360deg); }
}

/* Pulsar (agrandarse y reducirse) */
@keyframes pulsar {
  0%, 100% { transform: scale(1); }
  50%      { transform: scale(1.2); }
}

/* Aparecer desde abajo */
@keyframes aparecer {
  from { opacity: 0; transform: translateY(20px); }
  to   { opacity: 1; transform: translateY(0); }
}

Variables CSS (:root)

¿Qué son las variables CSS?

Permiten guardar valores (colores, tamaños, fuentes) en un solo lugar y reutilizarlos en todo el CSS. Para cambiar un color, solo se edita en :root.

/* Define las variables en :root (aplica globalmente) */
:root {
  --color-primario: #356630;
  --color-secundario: #2d2f3b;
  --color-texto: #000000;
  --fuente: 'Afacad', sans-serif;
  --radio-borde: 10px;
}

/* Usa las variables con var() */
.boton {
  background-color: var(--color-primario);
  font-family: var(--fuente);
  border-radius: var(--radio-borde);
}
💡 Convención: Los nombres de variables siempre empiezan con -- (doble guión).

Media Queries (Diseño Responsivo)

¿Qué son las media queries?

Aplican estilos CSS solo cuando se cumplen ciertas condiciones de pantalla. Hacen que el diseño se adapte a móviles, tablets y escritorios.

/* Se activa cuando la pantalla tiene máximo 480px (móvil) */
@media (max-width: 480px) {
  .menu {
    flex-direction: column; /* menú en columna en móvil */
  }
  .grid-proyectos {
    grid-template-columns: 1fr; /* 1 columna en móvil */
  }
}

/* Se activa cuando la pantalla tiene máximo 768px (tablet) */
@media (max-width: 768px) {
  .grid-proyectos {
    grid-template-columns: repeat(2, 1fr); /* 2 columnas en tablet */
  }
}

Breakpoints estándar

DispositivoBreakpointUso típico
Móvil pequeñomax-width: 480px1 columna, menú en columna
Tabletmax-width: 768px2 columnas, layout simplificado
Laptopmax-width: 1024px3 columnas
Escritoriomin-width: 1025pxLayout completo
⚠️ Importante: Las media queries van al FINAL del CSS para que sobreescriban los estilos normales correctamente.

Selectores y Especificidad CSS

Tipos de selectores

/* Selector de etiqueta - aplica a todos los p */
p { color: black; }

/* Selector de clase - aplica a class="boton" */
.boton { background: green; }

/* Selector de id - aplica a id="menu" */
#menu { display: flex; }

/* Selector descendiente - p dentro de .seccion */
.seccion p { font-size: 1rem; }

/* Selector hijo directo */
.menu > a { color: white; }

/* Pseudo-clase :hover */
.boton:hover { background: blue; }

/* Pseudo-clase :focus */
.campo:focus { border-color: green; }

Especificidad — quién gana cuando hay conflicto

SelectorPesoEjemplo
!importantGana siempre (evitar)color: red !important
id (#)100#menu
clase (.)10.boton
etiqueta1p, div, h1
💡 Regla: Si dos estilos apuntan al mismo elemento, gana el que tiene más especificidad. Si tienen el mismo peso, gana el que está más abajo en el CSS.

El Modelo de Caja (Box Model)

¿Qué es el Box Model?

Todo elemento HTML es una caja con 4 capas:

.caja {
  content:  /* el contenido real (texto, imagen) */
  padding:  20px;    /* espacio DENTRO del borde */
  border:   2px solid black; /* el borde */
  margin:   30px;    /* espacio FUERA del borde */

  /* box-sizing: border-box hace que padding y border
     se incluyan dentro del width total del elemento */
  box-sizing: border-box;
}

Margin y Padding — sintaxis corta

/* 1 valor: todos los lados iguales */
margin: 20px;

/* 2 valores: arriba/abajo  izquierda/derecha */
margin: 20px 40px;

/* 4 valores: arriba  derecha  abajo  izquierda (sentido horario) */
margin: 10px 20px 30px 40px;

/* Centrar horizontalmente un bloque */
margin: 0 auto;

Tipografía y Texto

Propiedades de texto principales

.texto {
  font-family: 'Afacad', sans-serif; /* fuente */
  font-size: 1rem;                 /* tamaño (relativo) */
  font-weight: 700;               /* grosor: 400=normal, 700=negrita */
  font-style: italic;             /* cursiva */
  line-height: 1.6;              /* espaciado entre líneas */
  letter-spacing: 2px;           /* espacio entre letras */
  text-align: center;            /* left | center | right | justify */
  text-decoration: none;          /* quita subrayado de enlaces */
  text-transform: uppercase;      /* MAYÚSCULAS */
  color: #333;                    /* color del texto */
}

Unidades de medida relativas vs absolutas

UnidadTipoEquivale a
pxAbsolutaPíxeles fijos — no escala
remRelativaRelativo al font-size del <html> (16px por defecto). 1rem = 16px
emRelativaRelativo al font-size del elemento padre
%RelativaPorcentaje del elemento padre
vwRelativa% del ancho de la ventana. 100vw = ancho completo
vhRelativa% del alto de la ventana. 100vh = alto completo
💡 Recomendación: Usa rem para tamaños de fuente y px para bordes. Evita px en font-size para mejor responsividad.

Colores en CSS

Formatos de color

/* Nombre */
color: red;
color: white;

/* Hexadecimal (el más usado) */
color: #356630;     /* verde oscuro */
color: #356630dc;   /* con transparencia (dc = ~86%) */

/* RGB */
color: rgb(53, 102, 48);

/* RGBA (con transparencia: 0=transparente, 1=opaco) */
color: rgba(53, 102, 48, 0.5);  /* 50% transparente */

/* Gradiente */
background: linear-gradient(135deg, #0d1f0a, #356630);

Seleccionar elementos del HTML

// Por id
const boton = document.getElementById('mi-boton');

// Por clase (devuelve el primero)
const tarjeta = document.querySelector('.tarjeta');

// Por clase (devuelve todos)

Eventos y acciones

// Al hacer clic en un botón
boton.addEventListener('click', function() {

  // Cambiar texto
  elemento.textContent = 'Nuevo texto';

  // Mostrar u ocultar
  elemento.style.display = 'block';  // mostrar
  elemento.style.display = 'none';   // ocultar

  // Cambiar estilo
  elemento.style.color = 'red';

  // Agregar o quitar una clase CSS
  elemento.classList.add('activo');
  elemento.classList.remove('activo');
  elemento.classList.toggle('activo'); // agrega si no está, quita si está
});
